We can do the following

Use <name> tag for <person> and <place>.
Use <rs> tag for <genre> <title> and <nationalCinema>

For example,
Instead of <person ref='gaidai'>Gaidai</person>
We will use TEI compliant 
<name type='person' ref='gaidai'>Gaidai</name>

Instead of <place ref="volgogradskayaOblast">Волгоградской области</place>
we will use
<name type='place' ref="volgogradskayaOblast">Волгоградской области</name>

Instead of <genre ref="skazka">сказка</genre>
we will use
<rs type="genre" ref="skazka">сказка</rs>

Instead of <title ref="elusiveAvengers>Неуловимые Мстители</title>
we will use
<rs type="title" ref="elusiveAvengers>Неуловимые Мстители</title>
